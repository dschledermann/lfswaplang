Swap language

On this branch select a language to swap for the default language.

This will swap the following fields on the following tables:

Pages and Pages language overlay

  • Title

Content

  • Header
  • Sub header
  • Bodytext

The uid's of each element is unchanged. This means that all relations are unchanged.

Other fields are not subject to swap. Contact support if your data set has content in other fields that needs swapping.
